# SOP: Monthly Support Plan

## Purpose
Ongoing support and maintenance for restaurants on the monthly plan ($199–$399/month).

## Service Tiers

| Feature | Essential ($199) | Premium ($399) |
|---------|-----------------|----------------|
| Menu updates (items, prices, modifiers) | Up to 10/month | Unlimited |
| Square dashboard monitoring | Weekly check | Daily check |
| Online ordering adjustments | 2/month | Unlimited |
| Monthly performance report | Basic | Detailed with recommendations |
| Priority support response | 24-hour | 4-hour (business hours) |
| On-site visit | Not included | 1/quarter |
| GBP/Facebook updates | Not included | Included |

---

## Monthly Checklist

### Week 1: Review & Report
- [ ] Pull Square sales report (last 30 days)
- [ ] Review: total orders, revenue, average order value
- [ ] Review: online vs. in-store order split
- [ ] Review: top-selling items, underperforming items
- [ ] Check for failed orders or customer complaints
- [ ] Note seasonal opportunities (upcoming holidays, events)

### Week 2: Maintenance
- [ ] Apply any menu changes requested by client
- [ ] Update trading hours (if seasonal changes)
- [ ] Check online ordering page loads correctly (mobile + desktop)
- [ ] Verify QR codes still scan correctly
- [ ] Check printer and hardware status with client

### Week 3: Optimisation
- [ ] Suggest menu changes based on data (hide low sellers, promote high margin)
- [ ] Review and respond to new Google reviews (Premium tier)
- [ ] Update Facebook page if needed (Premium tier)
- [ ] Check competitor activity (new restaurants, pricing changes)

### Week 4: Reporting
- [ ] Generate monthly report
- [ ] Send to client via email
- [ ] Schedule 15-min call to discuss (if client wants)
- [ ] Log any action items for next month

## Client Communication

### Monthly email (send with report)
Subject: `[Restaurant Name] — Square Monthly Report — [Month]`

Body:
> Hi [Name],
>
> Here's your monthly Square report for [Month].
>
> [Key highlight — e.g., "Online orders are up 15% this month."]
>
> See the attached report for full details. Happy to jump on a quick call if you'd like to discuss.
>
> Cheers,
> Newcastle Hub

### Escalation
- If client reports a critical issue (POS down, orders failing): respond within 1 hour
- If Square platform issue: check status.squareup.com, notify client, log ticket
